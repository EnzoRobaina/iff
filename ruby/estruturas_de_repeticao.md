### Estrutura de Repetição Usando FOR
```ruby
for contador in 0..10
  puts contador
end
```

### Estrutura de Repetição Usando WHILE
```ruby
contador = 0
while contador < 10
  puts contador
  contador += contador
end
```

### Estrutura de Repetição Usando EACH em Vetores
```ruby
vetor = [1,2,3]
vetor.each do |elemento|
  puts elemento
end
```
